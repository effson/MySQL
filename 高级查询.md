## 1. 基础查询
### 全部查询
```SQL
SELECT * FROM table_name;
```
### 查询部分字段
```SQL
SELECT `field1`, `field2` FROM table_name;
```
### 别名
```SQL
SELECT `field1` AS 'NAME1', `field2` AS 'NAME2' FROM table_name;
```
### 去除重复记录
```SQL
SELECT distinct `field1` FROM table_name;
```
## 2. 条件查询
```SQL
SELECT * FROM `table_name` WHERE `field` = 'value';
```
